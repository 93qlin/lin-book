~~常用来取整

~是按位非，就是每一位取反，
~~常用来取整
比如 ~~10.2323=10
~~（10/3） = 3

~~(Math.random() * 10) //4

更多： 是一种利用符号进行的类型转换,转换成数字类型

~~true == 1
~~false == 0
~~"" == 0
~~[] == 0
~~undefined ==0
~~!undefined == 1
~~null == 0
~~!null == 1
